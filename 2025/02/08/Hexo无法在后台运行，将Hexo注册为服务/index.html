Hexo无法在后台运行，将Hexo注册为服务

发表于2025-02-08|更新于2025-02-08
|阅读量:

首先创建服务，大概的步骤如下：

步骤 命令
创建服务文件 sudo nano /etc/systemd/system/<service>.service
重新加载 systemd 配置 sudo systemctl daemon-reload
设置开机启动 sudo systemctl enable <service>
启动服务 sudo systemctl start <service>
检查服务状态 sudo systemctl status <service>

首先需要将Hexo注册为服务，这样才能在后台保持运行
