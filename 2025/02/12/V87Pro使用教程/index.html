VGN客制化键盘V87Pro使用教程

发表于2025-02-12|更新于2025-02-12|原创文章外设
|阅读量:

VGN V87

VGN V87/V87PRO 是一款三模连接87配列的客制化机械键盘。

功能 快捷键
自定义灯光 Fn + Ins
关闭背光 Fn + Del
调节灯光亮度 Fn + ↑↓
调节灯光方向 Fn + ←
调节七色背光 Fn + →
调节背光速度 Fn + -=
显示电量 Fn + 右 Ctrl
切换Win/MAC系统 Fn + A/S
锁定Win键 Fn + Win
蓝牙对码 Fn + 123
2.4G对码 Fn + 4
恢复出厂设置 长按 Fn + Esc
长条指示灯灯效 Fn + Home
长条指示灯颜色切换 Fn + End
长条指示灯速度增加 Fn + PgUp
长条指示灯速度减小 Fn + PgDn
长条指示灯亮度调节 Fn + []
我的电脑 Fn + F1
浏览器 Fn + F2
计算器 Fn + F3
播放器 Fn + F4
上一首 Fn + F5
下一首 Fn + F6
暂停 Fn + F7
停止 Fn + F8
静音 Fn + F9
音量减 Fn + F10
音量加 Fn + F11
邮件 Fn + F12
